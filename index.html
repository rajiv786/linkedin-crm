Privacy Policy

Effective Date: June 28, 2026

LinkedIn CRM Saver ("the Extension") is designed to help users organize publicly visible information from LinkedIn pages into a structured contact list. We respect your privacy and are committed to protecting your information.

Information Processed

The Extension only processes information from the LinkedIn page that the user has intentionally chosen to extract.

This information may include publicly visible:

  • Name
  • LinkedIn profile URL
  • Job title
  • Company
  • Email address
  • Phone number
  • Location
  • Application links
  • Public post content

How Information Is Used

The extracted information is used solely to provide the Extension's functionality, including:

  • Creating a structured contact list
  • Searching and filtering contacts
  • Exporting contacts to CSV
  • Generating summaries using an AI provider selected by the user

Local Storage

Extracted contacts, user preferences and settings are stored locally using chrome.storage.local.

The developer does not receive, store or access this information.

AI Services

If the user configures a Groq API key, the Extension sends only the page content required for extraction to the Groq API using the user's own API credentials.

If the user enables Ollama, requests are sent only to the user's local Ollama instance running on their own computer (localhost).

The Extension does not provide its own remote AI service.

Data Sharing

The Extension does not:

  • Sell user data
  • Rent user data
  • Share user data with advertisers
  • Use analytics or tracking software
  • Collect browsing history

Permissions

The Extension requests only the permissions necessary to perform its intended function:

  • storage — Save contacts and user settings locally.
  • activeTab — Access the current LinkedIn page after user action.
  • scripting — Run the extraction script on the selected page.
  • tabs — Communicate with the active browser tab.
  • https://www.linkedin.com/* — Read LinkedIn pages selected by the user.

User Control

Users can delete all locally stored information at any time by clearing the Extension's data or uninstalling the Extension.

Children's Privacy

The Extension is not intended for children under the age of 13.

Changes to this Privacy Policy

This Privacy Policy may be updated from time to time. Any changes will be published on this page with an updated effective date.

Contact

For questions regarding this Privacy Policy, please contact the developer through the Chrome Web Store support page or the GitHub repository associated with this Extension.
